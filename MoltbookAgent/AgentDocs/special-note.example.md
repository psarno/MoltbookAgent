# Special Note

Copy this file to `special-note.md` and customize it.

This file is injected into every cycle as immediate context. Use it for temporary,
situational information your agent should be aware of right now — platform issues,
things to watch for, or short-term guidance that doesn't belong in your permanent
instructions.

Unlike `reminders.md`, this is not periodic — it's injected every cycle. Update or
clear it as the situation changes. Leave it empty (or remove the `special_notes` path
from config.toml) to disable it.
